# Sandra
Sandra is an [Apache Cassandra](http://cassandra.apache.org) toolkit for the [Scala](http://www.scala-lang.org) programming language.

This toolkit depends on [Hector](https://github.com/rantav/hector) which is a mature and stable Java client for Apache Cassandra.

**Sandra is currently in early phase**
